### 请按照下列每个声明指定的那样编写 x 类型的完整描述。
***
(a) char (\*x[10])(int);<br>
(b) int  (\*x(int))[5];<br>
(c) float \*(\*x(void))(int);<br>
(d) void (\*x(int, void (\*y)(int)))(int);
***

<br>

描述规则：（p.335 18.4.1）<br>
1. 始终从内往外读声明符。换句话说，定位声明的标识符，并且从此处开始解释声明。
2. 在作选择时，始终使 [] 和 () 优先于 \*。如果 \* 在标识符的前面，而标识符后面跟着 []，那么标识符表示数组而不是指针。同样的，如果 \* 在标识符的前面，而标识符后面跟着 ()，那么标识符表示函数而不是指针。（当然，可以使用圆括号来使 [] 和 () 相对于 \* 的优先级无效。）

<br>

参考答案：<br>
|题号|完整描述|
|:-:|:-|
|(a)|首先 x 是一个含有十个元素的数组，数组元素都是指针，指针指向的是函数，即函数指针，函数的返回值是 char 类型，形式参数有一个，类型为 int。|
|(b)|首先 x 是一个函数，函数的形式参数有一个，是 int 类型，函数返回指针，指针指向的是数组，数组有五个元素都是 int 类型。|
|(c)|首先 x 是一个函数，函数的形式参数要求为 void 类型，即不需要实际参数，函数会返回指针，该指针指向另一个函数，该函数的形式参数为 int 类型，并返回指向 float 类型的指针。|
|(d)|首先 x 是一个函数，该函数有两个形式参数，第一个形式参数是 int 类型，第二个形式参数是一个函数指针，该函数指针指向只有一个形式参数且参数类型为 int，返回值为 void，即无返回值的函数。x 的返回值为指针，该指针指向了另一个形式参数为 int 类型，返回值为 void 类型，即无返回值的函数，即函数指针。|
