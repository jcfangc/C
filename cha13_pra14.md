### 下面程序的输出是什么？
***
~~~
    #include <stdio.h>

    int main(void)
    {
        char s[] = "Hsjodi", (*p);

        for (p = s; *p; p++) {
            --(*p);
        }
        puts(s);
        return 0;
    }
~~~
***

<br>

参考答案：<br>
Grinch<br>
程序最主要的是循环部分的理解。循环中指针 p 遍历了字符串 s。遍历的循环中 p 指向的原始变量 -1。由于存在 ASCII，字符和数字可以有特别的映射关系。在本题中把字符串 s 中的每一个字符安装英文字母顺序向前推一位即可。顺便一提，Grinch 是圣诞怪杰的意思，一个卡通形象。
